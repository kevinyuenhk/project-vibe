# Project Vibe ⚔️

Isometric dungeon crawler built with **Godot 4.6** — inspired by 地城之光 (Dungeon & Fighter style).

## Controls (WIP)
| Key | Action |
|-----|--------|
| WASD / Arrow | Move |
| Mouse Click / Space | Attack |
| Shift | Dash |

## Setup
1. Install [Godot 4.6+](https://godotengine.org)
2. Open `project.godot` in Godot Editor
3. Press F5 to run

## Structure
```
scenes/    — .tscn scene files
scripts/   — GDScript
assets/    — sprites, audio, tilesets, fonts
```
